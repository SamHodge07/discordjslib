# discordjslib
A node.js Discord library!

Install: 
```
npm i discordjslib
```

Example:
```js
const Discord = require('discordjslib')
const client = new Discord().Client('token')
client.channels.find('channelid').then(channel => channel.send('Hello World!'))
```
