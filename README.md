# discordjslib
A node.js Discord library!


Install: 
```
npm i discordjslib
```


Example:
```js
const Discord = require('discordjslib')
const client = new Discord().Client('client-token')
client.channels.find('channel-id').then(channel => channel.send('Hello World!'))
```
